# Data Visualization Collection

## Overview

This repository contains a collection of data visualization exercises and assignments completed during my **Master of Science in Artificial Intelligence at the University of North Texas (UNT)**.

The purpose of this repository is to explore different visualization techniques, understand data patterns, and represent information effectively using various datasets.

This is **not a machine learning project**. It is a collection of visualization examples created as part of coursework and practice.

---

## Objectives

* Understand data visualization concepts.
* Explore different types of charts and visual representations.
* Analyze datasets through graphical exploration.
* Improve data interpretation and storytelling skills.

---

## Tools & Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook
* CSV datasets

---

## Visualizations Included

The repository contains visualizations created using different datasets:

### Datasets

* **IRIS.csv / Iris-Species.csv**

  * Exploratory visualization of the Iris dataset.
  * Analysis of flower features and species distribution.

* **Automakers.csv**

  * Visual analysis of automobile-related data.

* **Activity6_Population.csv**

  * Population-based data visualization.

* **PersonType.csv**

  * Visualization and analysis of categorical data.

* **bubblechart.csv**

  * Bubble chart-based visualization.

* **data.csv / Final.csv**

  * Additional datasets used for visualization practice.

---

## Visualization Techniques

Examples include:

* Bar Charts
* Scatter Plots
* Bubble Charts
* Histograms
* Distribution Plots
* Category-based Comparisons
* Data Exploration Charts
